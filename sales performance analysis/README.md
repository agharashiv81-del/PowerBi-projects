
# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview

The **Sales Performance Dashboard** is an interactive Power BI report designed to analyze sales performance across different categories, regions, customer segments, and shipping modes. The dashboard provides key business insights using KPIs and interactive visualizations to help users make data-driven decisions.

---

## 🚀 Features

- Interactive filters (Slicers)
- KPI Cards
- Professional dashboard design
- Business insights through visualizations
- Easy-to-understand layout

---

## 📈 Key Performance Indicators (KPIs)

- 💰 Total Sales
- 📦 Total Quantity
- 💵 Total Profit
- 📊 Average Sales
- 📉 Profit Margin %

---

## 📊 Dashboard Visuals

- Sales by Category
- Profit by Sub-Category
- Sales by Segment
- Sales by Region (Treemap)
- Profit by Ship Mode

---

## 🎛️ Filters

- Region
- City
- Segment
- Category
- Ship Mode

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dashboard is created using a retail sales dataset containing the following fields:

- Ship Mode
- Segment
- City
- State
- Region
- Category
- Sub-Category
- Sales
- Quantity
- Discount
- Profit

---

## 💡 Business Insights

- Technology category generates the highest sales.
- Consumer segment contributes the largest share of sales.
- Standard Class shipping generates the highest profit.
- Profit varies significantly across product sub-categories.
- Regional analysis helps identify high-performing sales regions.
