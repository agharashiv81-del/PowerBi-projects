# 📱 Mobile Sales Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze mobile phone sales performance across different cities, brands, payment methods, and time periods.

The dashboard provides business insights through Key Performance Indicators (KPIs), sales trends, customer ratings, payment analysis, and year-over-year comparisons.

---

## 🛠️ Tools & Technologies Used

- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📊 Dashboard Pages

### 1. Main Dashboard
- Total Sales Analysis
- Total Quantity Sold
- Total Transactions
- Average Price
- Sales by City
- Sales by Mobile Model
- Customer Ratings
- Payment Method Analysis
- Sales by Day Name

### 2. MTD (Month-to-Date) Report
- Daily Sales Performance
- Month-to-Date Tracking
- Trend Analysis
- Year and Month Filtering

### 3. Same Period Last Year Analysis
- Year-over-Year Sales Comparison
- Quarterly Sales Analysis
- Monthly Sales Comparison
- Historical Performance Tracking
